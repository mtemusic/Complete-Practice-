# MTE Music Practice Database

This is a static GitHub Pages-ready database app for the MTE Music Complete Practice Reference system.

## Files
- `index.html` — main app page
- `css/style.css` — styling
- `js/app.js` — client-side rendering logic
- `data/mte-complete-database.json` — the practice database
- `.nojekyll` — disables Jekyll processing on GitHub Pages

## GitHub setup
1. Create a public GitHub repository.
2. Upload all files and folders from this project.
3. Commit to the `main` branch.
4. In **Settings > Pages**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.

## Result
The app will load the JSON file directly in the browser and render the database as a static site.
